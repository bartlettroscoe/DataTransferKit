Given an entity and a reference point, evaluate the gradient of the shape function of the entity at that point. A default implementation is provided using a finite difference scheme.

Parameters

reference_pointEvaluate the shape function at this point given in reference coordinates.
gradientsEntity shape function gradients evaluated at the reference point. Return these ordered with respect to those return by getDOFIds() such that gradients[N][D] gives the gradient value of the Nth DOF in the Dth spatial dimension.
